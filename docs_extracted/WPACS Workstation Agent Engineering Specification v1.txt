WPACS Workstation Agent Engineering 
Specification v1.0 
Workforce Productivity & Activity Correlation System 
Endpoint Monitoring Agent Architecture & Engineering Standard 
 
1. DOCUMENT PURPOSE 
This document defines the architecture, behavior, deployment model, security requirements, 
event collection standards, and operational controls for the WPACS Workstation Agent. 
The Workstation Agent is the primary telemetry source for WPACS. 
Without this component: 
No workstation visibility 
 
No activity correlation 
 
No productivity calculation 
 
No state validation 
 
2. AGENT MISSION 
The agent must answer: 
Who is using the workstation? 
 
When are they using it? 
 
What state is the workstation in? 
 
Is the workstation active? 
 
Is the workstation idle? 
 
Is the workstation locked? 
 
Is the workstation disconnected? 
 
What applications are running? 

 
What workstation events occurred?
