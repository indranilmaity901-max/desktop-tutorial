WPACS API Contracts Specification v1.0 
Workforce Productivity & Activity Correlation System 
REST API Contract & Service Interface Specification 
Document Purpose 
This document defines the official API contracts for WPACS. 
These APIs serve as the integration layer between: 
 Frontend  
 Backend  
 Workstation Agent  
 TCN Integration Service  
 Rule Engine  
 Reporting Service  
 External Integrations  
 
1. API STANDARDS 
Protocol 
HTTPS Only 
 
API Style 
REST API 
 
Data Format 
{ 
  "success": true, 
  "data": {}, 
  "message": "", 
  "errors": [] 
} 
 

API Versioning 
/api/v1 
Examples: 
/api/v1/employees 
/api/v1/conflicts 
/api/v1/reports 
 
2. AUTHENTICATION API 
POST 
/api/v1/auth/login 
Request 
{ 
  "email": "admin@company.com", 
  "password": "********" 
} 
Response 
{ 
  "success": true, 
  "data": { 
    "access_token": "jwt_token", 
    "refresh_token": "refresh_token", 
    "expires_in": 3600 
  } 
} 
 
POST 
/api/v1/auth/refresh 
Request 
{ 
  "refresh_token": "token" 
} 
 

POST 
/api/v1/auth/logout

DELETE USER
